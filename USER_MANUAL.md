# Mass Start Scoring User Guide

## 1. Enter Race Results
1.  Go to the **Race Entry** tab.
2.  Select the **Race #** (e.g., 3 or 4) and **Gender**. Click **Load Form**.
3.  **Sprints**: Select the 1st, 2nd, and 3rd place winners for laps 4, 8, and 12.
4.  **Skaters**: For each skater, enter:
    *   **Status**: Usually "Finished". Change if they were Lapped, DNF, etc.
    *   **Pos**: Their finish position (1, 2, 3...).
    *   **Time**: Their official time (e.g., `8.15,33`).
5.  **Save**: Click the **Save Race Results** button at the bottom.
